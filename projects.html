Music Dance This is the game that created on November 2021 for the Advanced Multimedia subject coursework. The goal of this game is simple, collect nodes as much as possible while avoiding the bombs. This game make use of the CanvasRenderingContext2D API in rendering the scene. The entire game rendering are completely done through the JavaScript. HTML and CSS are used merely to provide the hosting for the canvas. Through the entire development process, the proper error handing has been the most critical part in order the ensure the good user experience. The development finished within a month. The game is currently hosted on https://greatkidshow.netlify.app/games/2. The source code is available on GitHub
